# deployment
## 部署系统定义
部署系统是一个管理软件包/配置、并按照一定规则将软件包与配置生效到主机的系统。

从系统构成视角看，部署系统由“包”/“配置”/“主机”对象构成。
从对象管理视角看，部署系统由“包仓库”/“配置中心”/“主机管理系统”+“部署逻辑管理系统”构成。

在后面章节中，我们逐个对象与管理系统拆解。
